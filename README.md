# Chapter 1. Getting started

Useful commands:

1. ```date```
2. ```w```
3. ```who```
4. ```who am i```
5. CTRL + U -- erases the whole input line.
6. CTRL + D -- kills the current process or even terminal. The same as ```exit```.
